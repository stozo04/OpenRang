# OpenLoop — Content rating (IARC) questionnaire answers

Transcribe into **Play Console → Policy → App content → Content ratings**. Google uses the IARC
questionnaire to assign regional ratings (ESRB, PEGI, USK, etc.). It's a Play Console web form, so
this file records the answers to enter.

> **Expected outcome:** the lowest rating tier everywhere — **ESRB Everyone / PEGI 3 / USK 0 / IARC 3+**.
> OpenLoop is a utility camera app with no objectionable content.

---

## Category

Choose the questionnaire category: **App (utility / productivity / tools)** — *not* a game.

## Answers

| Topic | Answer |
|---|---|
| Violence (cartoon, fantasy, realistic) | **No** |
| Sexual content / nudity | **No** |
| Profanity / crude humor | **No** |
| Controlled substances (drugs, alcohol, tobacco) | **No** |
| Gambling (simulated or real) | **No** |
| Fear / horror content | **No** |
| Discrimination / hate | **No** |
| Does the app share the user's current physical location with other users? | **No** |
| Does the app allow users to interact or exchange content with other users (chat, messaging, user-to-user content)? | **No** — OpenLoop has no in-app social/community features. Saving and the system **Share** action send the user's own clip to other apps they choose; there is no in-app user-to-user interaction. |
| Does the app allow users to purchase digital goods? | **No** |
| Is this a web browser / search engine? | **No** |
| Does the app contain or display user-generated content from a feed? | **No** |

## Other store-listing declarations that pair with the rating

| Declaration | Answer |
|---|---|
| Contains ads | **No** |
| In-app purchases | **No** |
| Target audience age groups | Recommended: **13+** (a general-audience tool; avoid declaring a child audience so the app isn't pulled into the Families program / additional child-data obligations — revisit only if you intend to target under-13). |
| Is the app designed for children? (Families policy) | **No** |

> The "users can interact / share content" question is the only one worth a second look. OpenLoop
> sharing goes **out** to other apps via Android's share sheet (no in-app community, feed, or
> user-to-user messaging), so **No** is the correct answer. If you ever add an in-app feed, comments,
> or messaging, re-take the questionnaire.
